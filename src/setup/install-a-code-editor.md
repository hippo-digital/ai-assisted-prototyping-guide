<span class="caption">Part 1: Get set up</span>
 <h1>Install a code editor</h1>

 <p>A code editor is where you will see the files in your prototype. You will also use its built-in terminal to run Gemini CLI.</p>
 <p>We recommend Visual Studio Code (VS Code). It is free and works on Mac, Windows, and Linux.</p>

 <h2>Step 1: Download VS Code</h2>
 <p>Go to <a href="https://code.visualstudio.com/" target="_blank" rel="noopener noreferrer">code.visualstudio.com</a> and download the version for your computer.</p>

 <h2>Step 2: Install it</h2>
 <p>Open the downloaded file and follow the installation instructions. Accept the default settings – you do not need to change anything.</p>

 <h2>Step 3: Open VS Code</h2>
 <p>Once installed, open VS Code. You should see a welcome screen.</p>

 <div class="inset-text">
 <p><strong>Checkpoint:</strong> You should now have VS Code open on your screen. If you see a welcome tab or an empty editor window, you are ready to move on.</p>
 </div>

 <h2>Step 4: Open the terminal in VS Code</h2>
 <p>You will use the terminal a lot in this guide, so it is worth learning how to open it now.</p>
 <p>In VS Code, press:</p>
 <ul>
 <li><strong>Mac:</strong> Ctrl + ` (the backtick key, usually above the tab key)</li>
 <li><strong>Windows:</strong> Ctrl + ' (the apostrophe key)</li>
 </ul>
 <p>Or go to the menu: <strong>Terminal</strong> → <strong>New Terminal</strong>.</p>
 <p>A panel should appear at the bottom of the VS Code window. This is your terminal.</p>

 <div class="inset-text">
 <p><strong>Checkpoint:</strong> You should see a terminal panel at the bottom of VS Code with a blinking cursor. It might show your username and computer name. This is normal.</p>
 </div>

 <details>
 <summary>The terminal looks intimidating</summary>
 <div class="details-body">
 <p>Yes. If you have never used a terminal before, it can feel strange to type commands instead of clicking buttons. That is completely normal.</p>
 <p>You do not need to learn terminal commands. This guide will tell you exactly what to type each time. Think of it like following a recipe – you do not need to understand the chemistry of baking to follow the steps.</p>
 </div>
 </details>

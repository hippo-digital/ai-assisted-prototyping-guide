<h1>AI-assisted prototyping with Gemini CLI</h1>

<p class="lead">Learn how to use Gemini CLI alongside the GOV.UK Prototype Kit to build realistic prototypes faster – even if you've never written a line of code.</p>

<div class="inset-text">
  <p>This guide is for designers and other professions who want to use AI tools to speed up prototyping. You do not need any coding experience.</p>
</div>

## What you'll learn

By the end of this guide, you'll have built a working GOV.UK prototype for a fictional 'Report a building issue' service – using plain English prompts instead of writing code by hand.

The guide has 2 parts:
